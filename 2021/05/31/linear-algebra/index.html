矩阵的逆

代数余子式:
$$
\sum_{k=1}^{n} a_{ik}A_{ik} = D\
\sum_{k=1}^{n} a_{kj}A_{kj} = D
$$
矩阵$A$可逆的冲要条件是$|A| \neq 0$

伴随矩阵：
$$A^* = \begin{pmatrix} A_{11},…,A_{n1}\\A_{12},…,A_{n2} \\…\\ A_{1n},…,A_{nn} \end{pmatrix}$$

$ A A^* = |A|I$，即$A^{-1} = \frac{1}{|A|} A^* $

  • $A$可逆， $Rank(AB) = Rank(BA) = Rank(B)$

线性方程

克莱默法则：

若$AX=B$,$|A| \neq 0$，则方程组必有唯一解

tips

  • 可以应用到齐次方程组的非零解中

齐次方程组$A_{m\times n}X =0$存在非零解时：$Rank(A) < n$

  • 定理：若$AX = 0$的系数矩阵的秩$R(A) = r < n$则$AX=0$有$t = n-r$个线性无关的解，且$AX= 0$的任一个解都可由这$t = n-r$个线性无关的解线性表示。
  • 定理： 若$\eta_{1},…,\eta_{t}$是$AX = 0$的基础解系，则齐次方程组$AX=0$的通解为：$$k_{1} \eta_{1} + k_{2} \ eta_{2} + … + k_t \eta_{t}$$其中$k$是任意常数。
  • 基础解系: $AX=0$的极大线性无关解。（不唯一）
  • 向量$\alpha_{1},\alpha_{2},…,\alpha_{n}$线性相关，则齐次方程组$(\alpha_{1} \alpha_{2} … \alpha_{n})X = 0$有非零解。
  • 若$\beta$可由$\alpha_{1},\alpha_{2},…,\alpha_{n}$线性表示，则非齐次方程组$(\alpha_{1} \alpha_{2} … \alpha_{n})X= \beta$有解。

方程的求解与应用

非齐次线性方程 $AX = B$有解$\Leftrightarrow R(A) = R(\overline{A})$其中$\overline{A}$为增广矩阵。

  • 有唯一解：$R(A) = R(\overline{A}) = n$
  • 无穷多解：$R(A) = R(\overline{A}) < n$

设$\alpha$是方程组$AX=B$的解，$\eta_1, \eta_2,…,\eta_t$是$AX=0$的基础解系，则方程组$AX=B$的通解为

$$\alpha + k_{1} \eta_{1} + … +k_{t} \eta_{t}$$

如果$\alpha_{1},\alpha_{2}$是$AX=B$的解，故$\alpha_{1} - \alpha_{2}$是$AX=0$的解。而$\alpha + \eta$是$AX=B$的解。

特征值，特征向量

设$A$是$n$阶矩阵，$\alpha$是$n$维非$0$列向量，且
$$
A\alpha = \lambda \alpha
$$
则称$\lambda$是矩阵$A$的特征值，$\alpha$是矩阵$A$对应子特征值$\lambda$的特征向量。

性质：
$$
|\lambda E - A| = 0
$$
故齐次线性方程有非零解，并且可以通过其求解特征值$\lambda_{i}$共$n$个，（含重根）。
$$
(\lambda_{i}E - A) X = 0
$$

找特征值的时候灵活运用代数余子式的性质，找特征向量的时候可以将矩阵的一行写成0

  • $(A + nE)\alpha = (\lambda + n)\alpha$
  • $A^{n} \alpha = \lambda^{n} \alpha$
  • $A^{-1} \alpha = \frac{1}{ \lambda } \alpha$
  • $A^* \alpha = \frac{|A|}{ \lambda } \alpha$

定理:

  1. 如果$\alpha_{1},\alpha_{2}$都是矩阵$A$对应于特征值$\lambda$的特征向量，则当$k_{1}\alpha_{1}+k_{2}\alpha_{2} \neq 0$，时，$k_{1}\alpha_{1}+k_{2}\alpha_{2} $仍是$A$关于特征值$\lambda$的特征向量。

  2. 如果$\lambda_{1}$与$\lambda_{2}$是$A$不同的特征值，对应的特征向量分别是$\alpha_{1}$和$\alpha_{2}$，则$\alpha_{1},\alpha_{2}$必线性无关

  3. $n$阶矩阵$A$，特征值是$\lambda_{1},…,\lambda_{n}$，则

    (1) $\sum \lambda_{i} = \sum a_{ii}$，即$tr(A)$，可用韦达定理证明。证明

    (2) $|A| = \prod \lambda_{i}$

相似矩阵

$$
A \sim B: \exists 可逆P, P^{-1}AP=B
$$

如果$A \sim \Lambda $，则称$A$可相似对角化。

定理:

  • $A \sim \Lambda , \Longleftrightarrow , A$有$n$个线性无关的特征向量。证明

故有$\Lambda$为特征值$\lambda$组成的对角矩阵，而可逆初等变换矩阵$P$为$A$的特征向量组成。

  • $A \sim \Lambda \Longleftrightarrow $ 若$\lambda $是$A$的$k$重特征值，则$\lambda$有$k$个线性无关的特征向量。

性质：

  • $|\lambda E - A | = | \lambda E - B | \Rightarrow \lambda_{A} = \lambda_{B}$
  • $|A| = |B|$

实对称矩阵

性质：

  • 一定可以相似对角化，即$A \sim \wedge ,\wedge$为对角矩阵。
  • 特征值不同的特征向量互相正交。
  • 可用正交矩阵相似对角化。
