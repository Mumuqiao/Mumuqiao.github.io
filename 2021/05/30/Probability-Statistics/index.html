泊松分布

$$X\sim B(n,p)$$

等待型分布

即第一次中即停止，投篮投到第一个中就停止

几何分布（离散型）

$$X \sim G ( p ) $$

指数分布（连续型）

$$E(\lambda), {\lambda} > 0$$

$$f(x) = \begin{cases} \lambda e^{- \lambda x} , &x>0 \\ 0,&else \end{cases}$$

$$F(x)=\begin{cases} 1- e^{- \lambda x}, & x \geq 0 \\ 0 , &x<0 \end{cases}$$

期望

$$ E(X) = \frac{1}{ \lambda } $$

$\lambda$代表失效率，即期望代表着平均等待时间。即失效率高，机器容易坏，等待坏的时间短。故$\lambda > 0$

tips

  • 当事件服从$X \sim E(\lambda)$时，等待了$t$小时的情况下，在等待$a$小时后事件发生的概率等于直接等待$a$小时的概率。
  • 有分布不一定存在期望

泊松分布

$$X\sim P(\lambda)$$

$$P{X=k} = \frac{ \lambda ^{k}} {k!} e^{- \lambda }$$

期望

$E(X) = \lambda$

tips

  • 泊松分布代表着某场合某单位时间内源源不断的质点来流的个数$(k)$。故$k >0$
  • $\lambda$代表强度，即平均值。故$\lambda > 0$
  • 有时候可表示稀有事件发生的概率，即问在一盘青椒炒肉丝中存在100根肉丝的概率。

正态分布

$$X\sim N({\mu},{\sigma}^2)$$

$$f(x) = \frac{1}{\sqrt{2 \pi } \sigma }e^{-\frac{1}{2}(\frac{x- \mu }{ \sigma })^2},(-\infty<x<+\infty)$$

标准正太分布

$$ \varphi (x) = \frac{1}{\sqrt{2 \pi } \sigma }e^{- {\frac{x^{2}} {2}} }$$

分布函数$\Phi(X)$

只有标准正太函数用$\varphi$和$\Phi$

tips

  • 在$x={\mu}-{\sigma}$ 和$x = \mu + \sigma$处是函数凸到凹和凹到凸函数的分界点。
  • $\Phi(0) = \frac{1}{2}$ ，$\Phi(-X) = 1- \Phi(X)$
  • $F(x) = P{X \leq x} = \Phi(\frac{x-\mu}{\sigma})$
  • $ aX+b \sim N(a\mu + b, a^{2} \sigma^{2}),(a \neq 0)$

多维随机变量的概率分布

$Z = XY$概率密度函数$f(x,y)$，则Z的函数分布为：

$$F(Z) = \int_{-\infty}^{+\infty} \frac{1}{|x|}f(x, \frac{z}{x})dx$$

  • 口诀：

    积谁不变谁，换完求偏导(对$Z$,且加绝对值！)

当$X$和$Y$分别为离散和连续时，设$X$离散$Y$连续，$U =X+Y$，则:

$$
\begin{align*}
G(u) = P\{U \leq u\} &= P(X+Y \leq u) \\
&= P\{X+Y \leq u, \Omega\} \\
&= P\{X+Y \leq u, \{X=1\} \cup \{X = …\} \} \\
&= P\{ (X+Y\leq u, X = 1) \cup (X+Y \leq u, X=2)\} \\
&= P_1 + P_2
\end{align*}
$$

期望

  • 性质：
    1. 和差的期望无条件打开：$ E( \sum_{i=1}^{n} { a_{i} X_{i} } ) = \sum_{i=1} ^{n}{ a_{i} EX_{ i } }$
    2. $E(aX+c) = aEX+c$， $Ec = c$
    3. 乘除变量独立时打开: $ E [ \prod_{i=1}^{n} {g_{i} ( X_{i} ) } ] = \prod_{i=1}^{n} {E[ g_{i} ( X_{i} ) ] }$
    4. $E(g(X,Y)) = \int \int g(x,y)f(x,y)dxdy$

方差

$$
DX=E[(X-EX)^{2}] = E(X^{2}) - (EX)^{2}
$$

  • 性质：
    1. $D(aX+b) = a^{2}Dx$
    2. $D(X \pm Y) = DX + DY \pm 2Cov(X,Y)$
    3. $D(X) = Cov(X,Y)$

切比雪夫不等式

$$
P{|X-EX| \geq \epsilon } \leq \frac{DX}{\epsilon^{2}}, or \
P{|X-EX| < \epsilon } \geq 1- \frac{DX}{\epsilon^{2}}
$$

  • 【注】由切比雪夫不等式知，当$DX$愈小时,概率$P{|X一EX|< \epsilon }$愈大，这表明方差是刻画随机变量与其期望值偏离程度的量，是描述随机变量$X$“分散程度”特征的指标。
  • 应用：(1). 可以估算随机变量在某范围取值的概率；(2). 可以证明某些收敛性问题。

协方差

​ 如果随机变量$X$与$Y$的方差存在且$DX>0$，$DY>0$，则称$E[(X-EX)(Y-EY)]$为随机变量$X$与$Y$的协方差，并记为$Cov(X,Y)$，即
$$
Cov(X,Y) = E[(X-EX)(Y-EY)] = E(XY)-EX\times EY
$$
其中
$$
E(XY) =
\begin{cases}
\sum_{i} \sum_{j} x_{i}y_{j} P \{ X=x_{i},Y = y_{i}\},离散型 \\
\int_{- \infty}^{+ \infty} \int_{- \infty}^{+ \infty} xy f(x,y)dxdy,连续型
\end{cases}
$$
称
$$
\rho_{xy} = \frac{Cov(X,Y)}{\sqrt{DX} \sqrt{DY}}
$$
为相关系数。

性质

  • 单列可拆性：$Cov(\sum_{i=1}^{n} aX_{i}, Y) = \sum_{i=1}^{n} a_{i}Cov(X_{i},Y)$，$X$和$Y$一次只能拆一个，不能同时拆。
  • $Y= aX+b$，则$\rho_{XY} = {1, a>0 ; -1, a<0}$-

大数定律与中心极限定理

依概率收敛

​ 设随机变量$X$与随机变量序列${X_{n}}(n = 1,…,n)$，如果对任意$\epsilon >0$，有
$$
\lim_{n \rightarrow + \infty} P{|X_{n} - X| \geq \epsilon } = 0 或 \lim_{n \rightarrow + \infty}{|X_{n} - X| < \epsilon } = 1
$$
则称随机变量序列${X_{n}}$依概率收敛于变量X，记为
$$
\lim_{n \rightarrow + \infty} X_{n} = X ( P ) 或 X_{n}\xrightarrow{P} X,(n \rightarrow + \infty)
$$

大数定律

 在满足一定的条件下，所有的大数定律均为$\frac{1}{n} \sum_{i=1}^{n} X_{i} \xrightarrow{P} E(\frac{1}{n}\sum_{i=1}^{n}X_{i})$ ,即均值依概率收敛到均值的期望

  • 即均值是趋于稳定的。

切比雪夫大数定律

​ 假设${X_{n}}(n=1,2,…)$是相互独立的随机变量序列，如果方差$DX_{i},(i \geq 1)$存在且一致有上界，即存在常数$C$，使$DX_{i} \leq C$对一切$i \geq 1$均成立，则${X_n}$服从大数定律：
$$
\frac{1}{n} \sum_{i=1}^{n} X_{i} \xrightarrow{P} \frac{1}{n} \sum_{i=1}^{n} EX_{i}
$$
可以发现$\frac{1}{n} \sum_{i=1}^{n}EX_{i} = E(\frac{1}{n}\sum_{i=1}^{n}X_{i})$，即均值依概率收敛到均值的期望。

伯努利大数定律

​ 假设$\mu_{n}$是$n$重伯努利试验中时间$A$发生的次数，在每次实验中$A$发生的概率为$p(0 < p < 1)$，则$\frac{\mu_{n}}{n} \xrightarrow{P} p$，即对任意$\epsilon > 0$，有
$$
\lim_{n \rightarrow \infty} P \lbrace |\frac{\mu_{n}}{n} -p | < \epsilon \rbrace = 1
$$
即为抛硬币实验的描述。

辛钦大数定律

​ 假设${X_{n}}$是独立同分布的随机变量序列，如果$EX_{i} = \mu (i = 1,2,…)$存在，则$\frac{1}{n}\sum_{i=1}^{n} X_{i} \xrightarrow{ P } \mu$，即对任意$\epsilon >0$，有
$$
\lim_{n\rightarrow \infty}\lbrace|\frac{1}{n} \sum_{i=1}^{n} X_{i} - \mu| < \epsilon \rbrace = 1
$$
即切比雪夫大数定律在$X_{i}$独立同分布的情况下的形式。

中心极限定理

中心极限定理均为：若$X_{i}$服从独立同分布，$X_{i} \sim F, EX_{i} = \mu ,DX = \sigma^{2} $ ，则$ \sum_{i=1}^{n} X_{i} \sim N(n \mu, n \sigma^{2}),n \to \infty $

列维-林德伯格定理

​ 假设${X_{n}}$是独立同分布的随机变量序列，如果
$$
EX_{i} = \mu, DX_{i} = \sigma^{2} > 0(i=1,2,…)
$$
存在，则对任意实数$x$，有
$$
\lim_{n \rightarrow \infty}P \lbrace\frac{\sum_{i=1}^{n} X_{i} - n \mu}{\sqrt{n} \sigma} \leq x \rbrace = \frac{1}{\sqrt{2\pi}} \int_{- \infty}^{x} e^{- \frac{t^{2}}{2}}dt = \Phi(x)
$$

棣莫弗-拉普拉斯定理

​ 假设随机变量$Y_{n} \sim B(n,p)(0<p<1, n \geq 1)$，则对任意实数$x$，有
$$
\lim_{n \to \infty}P \lbrace \frac{Y_{n} - np}{\sqrt{np(1-p)} } \leq x \rbrace = \frac{1}{\sqrt{2\pi} } \int_{- \infty}^{x} e^{- \frac{ t^{2}}{2} }dt = \Phi(x)
$$

数理统计

简单随机抽样即样本独立同分布

统计量

​ 设$X_{1},X_{2},…,X_{n}$为来自总体$X$的一个样本，$g(x_{1},x_{2},…,x_{n})$为$n$元函数，如果$g$中不含任何未知参数，则称$g(X_{1},X_{2},…,X_{n})$为样本$X_{1},X_{2},…,X_{n}$的一个统计量。若$(x_{1},x_{2},…,x_{n})$为样本值，则$g(x_{1},x_{2},…,x_{n})$为$g(X_{1},X_{2},…,X_{n})$的观测值。

  • $g(X_{1},X_{2},…,X_{n})$的结果是一个随机变量$X$，即统计量。
  • $g(x_{1},x_{2},…,x_{n})$的结果为一个值。

数字特征

  • 样本方差 $S^{2} = \frac{1}{n-1} \sum_{i=1}^{n}(X_{i} - \overline{X})^{2}$
  • 样本$k$阶原点矩 $ A_{k} = \frac{1}{n} \sum_{i=1}^{n} ( X_{i} - 0 )^{k} = \frac{1}{n} \sum_{i=1}^{n} X_{i}^{k}, (k = 1,2,…)$
  • 样本$k$阶中心矩 $B_{k} = \frac{1}{n}\sum_{i=1}^{n} (X_{i} - \overline{X})^{k},(k = 2,3,…)$
  • $E\overline{X} = EX = \mu$
  • $D\overline{X} = \frac{1}{n}DX = \frac{\sigma^{2}}{n}$
  • $E(S^{2}) = DX = \sigma^{2} $

顺序统计量

​ 将样本$X_{1},X_{2},…,X_{n}$的$n$格观测量按其取值从小到大的顺序排列，得
$$
X_{(1)} \leq X_{(2)} \leq … \leq X_{(n)}
$$
随机变量$X_{(k)},(k \in Z^{+})$称作第$k$顺序统计量

  • $X_{(k)}$的分布函数为：$F_{ ( k ) } (x) = \sum_{i=k}^{n} C_{n}^{i} ( F ( x ) )^{i} ( 1-F( x ) )^{n-i} $
  • $X_{(k)}$的概率密度为：$f_{(k)}(x) = k C_{i=k}^{n} ( F( x ) )^{ ( k-1 ) } ( 1-F( x ) )^{n-k} f(x)$
  • $F_{(n)}(x) = [F(x)]^{n}$
  • $F_{(1)}(x) = 1 - [1-F(x)]^{n}$
