Pairwise registration

ICP algorithm(Iterative Closest Points)

  • Minimize the error function:

    $$E = \sum_{i}( R p_{i} + t - 1 _{i} )$$

    Iterate to find alignment.

  • The registration problem shall be formulated in a least squares sense as follows. Compute the rigid body transformation $\alpha *$, which minimizes

    $$F( \alpha ) = \sum_{i} d^{2} ( \alpha (x_{i}^{0} ), \Phi )$$

    Here, $d^{2} ( \alpha (x_{i}^{0} ), \Phi )$ denotes the squared distance of $\alpha (x_{i}^{0} )$ to $\Phi$.

  • Point-2-plane distance

    While using Gauss-Newton to deal with ICP, using point-2-plane distance instead of p2p allows flat regions slide along each other. If the two point clouds are in a good position, we can use point-2-plane, and vice versa.

    Point-2-plane Squared distance function

    $$F_{d}(x_{1} , x_{2} , x_{3} ) = \frac{d}{d-\varrho_{1} } x_{1}^{2} + \frac{d}{d-\varrho_{2} } x_{2}^{2} + x_{3}^{2} $$

    $\varrho$ is the inverse curvature, and $d$ is the distance. And using Newton method to deal with it.

    Geometry of the Squared Distance Function to Curves and Surfaces H. Pottmann, M. Hofer

Partial overlap

  • Optimization strategy 1: Gauss-Newton optimization

  • Optimization strategy 2: reweighted least squares
    $$
    min_{R, t} \sum_{i=1}^{N} w_{i} | (R p_{i} + t - f_{i})^{T} n_{i} |^{2}
    $$
    where
    $$
    w_{i} = \frac{ \rho ( | (R p_{i} + t - f_{i})^{T} n_{i} | )}{ | (R p_{i} + t - f_{i})^{T} n_{i} |^{2} }
    $$
    $\rho$ is a function that define the $w_{i}$, such as $\rho(x) = 1$.

Other feature

RANSAC

​ While we get a lot of co-respondent-pairs, how can we find that which one is good or bad?

​ We can solve this problem through rigid transform.

​ Here is the algorithm flow

  • Preprocessing: sample each object

  • Recursion:

    • Step 1: Sample three(two, three to 3D, two to 2D) pairs, check distance constraints
    • Step 2: Fit a rigid transform
    • Step 3: Check how many point pairs agree. If above threshold, terminates; otherwise goes to Step 1.

RANSAC+: points with normal, that only need two points in each transform.

Other approach

  • Hough transform: choose a points-pair and vote .

    Partial and Approximate Symmetry Detection for 3D Geometry, N. Mitra, L.Guibas, and M.Pauly,SIGGRAPH’06

  • Spectral approach: using a consistency matrix and check the clique constraint

    A Spectral Technique for Correspondence Problems using Pairwise Constraints,M. Leordeanu and M.Hebert, ICCV 2005

  • Spectral matching + reweighted least squares.

Learning-based methods

Multiple method

​ Very similar to ICP method.

Fix a scan and use other scans to find the minimum distance.

Joint pairwise registration Tam, Gary KL et al. 2012

Limitations:

  • Need to determine overlapping scans and overlapping regions

    ​ Potentially a quadratic number of pairs

  • Slow convergence when there are a lot of scans

Simultaneous registration and reconstruction

Build a latent surface and fix it by point cloud. after that we can reconstruct the whole model.

  • Step 1: Latent surface creation

    Fit planes to points associated with each cell

  • Step 2: Scan-surface alignment

It is equivalent to making an ICP for the latent surface in each scan
